QUICK START
===========

1. Build the compiler:
   cd flux_compiler
   cargo build --release
   cd ..

2. Compile a program:
   ./flux_compiler/target/release/fluxc compile main.fsh -o program

3. Execute:
   ./program

4. Read the documentation:
   - README-USAGE.md for getting started
   - COMPLETION-REPORT.md for technical details
   - INDEX.md for documentation navigation
