## ✅ What You Can Do With FluxSharp

✅ **Do:**
- Define functions and call them
- Use if/else statements
- Loop with while
- Work with integers and floats
- Create classes with properties/methods
- Perform math calculations
- Print output
- Combine operations

❌ **Cannot Yet (v1.0):**
- Use return values in functions
- Use for loops (use while instead)
- Create arrays/lists
- Use pointers/references
- Call functions recursively
- Allocate dynamic memory
- Use generics

## ⚡ Quick Syntax Reference

**Variables:**
```flux
int x = 42;
float f = 3.14f;
double d = 2.71828;
string s = "text";
bool b = true;
```

**Functions:**
```flux
void function_name(int param) {
    // code
}
```

**Classes:**
```flux
class ClassName {
    public int property;
    public void method() { }
}
```

**Control Flow:**
```flux
if (condition) { }
while (condition) { }
```

**Math:**
```flux
int sum = a + b;
double root = sqrt(16);
double pi = PI;
```
